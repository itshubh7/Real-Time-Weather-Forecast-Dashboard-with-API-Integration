🌦️ Real-Time Weather Forecast Analytics Dashboard

An interactive Power BI dashboard that analyzes real-time weather data using API integration. The report automatically refreshes data daily and provides insights into temperature trends, Air Quality Index (AQI), and multi-city weather comparisons.

📊 Project Overview

The Weather Forecast Report is a Power BI data visualization project built using live API data. The dashboard displays weather insights for six cities, allowing users to monitor temperature, AQI, and past week weather trends through interactive visuals and slicers.

🚀 Key Features

Real-time weather data integration using API

Automatic daily data refresh

Interactive single-page Power BI dashboard

Temperature trend analysis (past week)

Air Quality Index (AQI) monitoring

City-wise comparison using slicers
